# Examples

Live, self-contained artifacts the plugin generates. Open the HTML from disk, or use the hosted link.

| Example | What it teaches | Live |
|---|---|---|
| [`features-as-dimensions.html`](features-as-dimensions.html) | A feature is one measured dimension; an example is the point its features pin down. Five sliders, one face, one classifier. | [open](https://nagisanzenin.github.io/engram/examples/features-as-dimensions.html) |

Each obeys the seven-clause [Explorable Contract](../skills/_shared/explorable-contract.md): nothing unlocks without a committed prediction, the manipulable model is guided (predict → act → explain, with a worked drive first for novices), retrieval prompts are embedded, decoration is zero, and it ends by asking you to close the page and rebuild the argument from nothing.

Why those rules and not prettier ones: [`docs/06-visual-encoding.md`](../docs/06-visual-encoding.md).
